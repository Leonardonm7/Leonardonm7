## Featured Project

### [FR Editor](https://github.com/Leonardonm7/fr-editor)

A local-first mobile app for creating, organizing and running gym workout plans.

- Exercise library in Brazilian Portuguese.
- Workout planning by weekday.
- Active workout session with rest timer.
- Local persistence with SQLite.
- JSON backup import/export.
- Light/dark theme and font size settings.
